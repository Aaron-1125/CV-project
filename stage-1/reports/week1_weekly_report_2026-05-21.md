# 第一周周报：阶段一人脸识别基础与数据集探索

周期：第一周，2026-05-18 至 2026-05-21

## 本周已完成

- 完成阶段一开发环境与交付目录整理，保留 `demo/` 作为代码入口，`reports/` 作为报告和结果产物目录，`reports/assets/` 按用途分层存放图片，`data/` 与 `checkpoints/` 仅存放本地数据、缓存和模型权重。
- 完成人脸识别基础概念梳理，覆盖人脸检测、关键点定位、人脸对齐、embedding 提取、验证与识别等核心流程。
- 完成 CelebA 与 LFW 数据集探索。CelebA 共 `202599` 张图像、`10177` 个身份、`40` 个属性；LFW 共 `13233` 张图像、`5749` 个身份、`6000` 对 10-fold verification pairs。
- 使用 MMDetection GroundingDINO 开放词模型完成公开 LFW 测试图人脸检测，4 张测试图均检测到 `1` 张脸，并输出标框图和 JSON 摘要。
- 使用 InsightFace `buffalo_l` 完成关键点定位和 LFW 验证评估，4 张公开测试图均检测到 `1` 张脸；LFW 验证 `6000/6000` 对有效，10-fold mean accuracy 为 `0.9665`，AUC 为 `0.989726`。
- 更新 `README.md`、`environment.md` 与阶段报告，补齐运行命令、产物路径、公开测试图说明和验收结果。
